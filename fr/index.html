01. Profil

Bonjour ! Je m'appelle Tristan Berger, je suis un ingénieur de 24 ans qui aspire à devenir développeur full-stack.

J'ai un an d'expérience en tant qu'ingénieur en électronique, mais en septembre 2021, j'ai décidé de quitter ce domaine qui ne me passionnait pas pour me lancer dans le développement web. J'ai commencé en autodidacte, en suivant des cours en ligne et en créant mes propres projets.

J'ai choisi de me lancer dans cette reconversion car j'ai toujours adoré coder, je sens que le développement web fait partie de mon univers et que c'est un milieu dans lequel je vais m'épanouir. Je me sens à l'aise quand je développe et je suis extrêmement motivé à l'idée d'apprendre et de progresser pour un jour exceller dans ce domaine.

02. Compétences

Depuis le début de mon parcours, j'utilise des ressources en ligne comme openClassrooms, Codecademy etc... pour apprendre les langages du web, mais j'ai aussi suivi des cours sur les fondamentaux du web, le SEO et l'accessibilité qui étaient des notions que je ne connaissais que très peu.

Mes expériences passées m'ont permis d'avoir des bases solides en programmation, cela a donc facilité l'apprentissage de HTML/CSS/JS/React et PHP.

Pour développer mes compétences autant que possible, j'ai complété des challenges FrontendMentor, et j'ai réalisé quelques projets personnels.

03.Quelques projets

Clone Twitter

Un clone du réseau social Twitter en PHP. L'utilisateur à accès à de nombreuses fonctionnalités de twitter (tweet, retweet, like, follow, citations, mentions notifications etc..). Ce projet est un véritable challenge de part sa complexité, mais également car c'est ma première expérience avec le langage PHP. J'ai décidé de ne pas utiliser de framework afin de me familiariser avec ce nouveau langage.

  • SASS
  • JS
  • PHP
  • MySQL
  • MVC architecture

Clone Netflix

Un clone de Netflix en react, permettant de voir les bandes-annonces et les informations importantes des films. Firebase est utilisé pour l'authentification, et les films sont tirés de l'API The Movie Database. Les utilisateurs peuvent filtrer les films par catégorie, mais aussi rechercher n'importe quel film disponible dans la base de données en utilisant la barre de recherche.

  • SASS
  • React
  • API
  • Firebase

Mini-jeu Clofus

Un mini-jeu codé en Javascript grandement inspiré du MMORPG Dofus. Le jeu comprend une carte d'accueil dans laquelle l'utilisateur peut se déplacer librement grâce à l'implémentation de l'algorithme de pathfinding Dijkstra. Ensuite, l'utilisateur a accès à un combat tactique assez complexe. Afin d'obtenir un objet très puissant, il doit traverser un labyrinthe protégé par un monstre.

  • SASS
  • JS

FrontendMentor Challenge

Space tourism

Un défi proposé par Frontend Mentor. Le défi consistait à construire un site web de tourisme spatial de plusieurs pages et à le rendre aussi proche que possible du design. J'ai découvert la puissance des utility classes et des custom properties. Le site respecte les spécifications ARIA, est entièrement responsive et utilise JavaScript pour modifier dynamiquement le contenu.

  • SASS
  • JS
  • Custom properties
  • Utility classes

Portfolio

Le site sur lequel vous vous trouvez, réalisé sans framework, en anglais et français, avec redirection en fonction du pays de l'utilisateur. Ce site est entièrement accessible, il respecte les spécifications ARIA et il est optimisé pour le SEO ( testé avec lighthouse ). Firebase est utilisé pour le back-end, notamment sa realtime database pour stocker les données du formulaire de contact, ainsi que les clouds functions pour m'envoyer un mail lors de la modification de la database ( nodemailer avec protocole OAuth2.0 ).

  • SASS
  • JS
  • Firebase
