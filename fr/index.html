01. Profil

Bonjour ! Je m'appelle Tristan Berger, je suis un ingénieur de 24 ans qui aspire à devenir un développeur full-stack.

J'ai un an d'expérience en tant qu'ingénieur en électronique, mais en septembre 2021, j'ai décidé de me lancer dans le développement web. J'ai commencé en autodidacte, en suivant des cours en ligne et en créant mes propres projets.

Je sens que le développement web fait partie de mon univers, je me sens à l'aise quand je développe, et surtout, j'aime ce que je fais.

02. Compétences

Depuis le début de mon parcours, j'utilise des ressources en ligne comme openClassrooms, Codeacademy etc... pour apprendre les langages du web, mais j'ai aussi suivi des cours sur les fondamentaux du web, le SEO et l'accessibilité qui étaient des notions que je ne connaissais pas beaucoup.

J'avais l'habitude de développer en C/C++, cela a donc facilité l'apprentissage de HTML/CSS/JavaScript et React.

Pour développer mes compétences autant que possible, j'ai complété des challenges FrontendMentor, et j'ai réalisé quelques projets personnels.

03.Mes projets

Clone Netflix

Un clone de Netflix en react, permettant de voir les bandes-annonces et les informations importantes des films. Firebase est utilisé pour l'authentification, et les films sont tirés de l'API The Movie Database. Les utilisateurs peuvent filtrer les films par catégorie, mais aussi rechercher n'importe quel film disponible dans la base de données en utilisant la barre de recherche.

  • SASS
  • React
  • API
  • Firebase

Mini-jeu Clofus

Un mini-jeu codé en Javascript grandement inspiré du MMORPG Dofus. Le jeu comprend une carte d'accueil dans laquelle l'utilisateur peut se déplacer librement grâce à l'implémentation de l'algorithme de pathfinding Dijkstra. Ensuite, l'utilisateur a accès à un combat tactique assez complexe. Afin d'obtenir un objet très puissant, il doit traverser un labyrinthe protégé par un monstre.

  • SASS
  • JS

FrontendMentor Challenge

Space tourism

Un défi proposé par Frontend Mentor. Le défi consistait à construire un site web de tourisme spatial de plusieurs pages et à le rendre aussi proche que possible du design. J'ai découvert la puissance des utility classes et des custom properties. Le site respecte les spécifications ARIA, est entièrement responsive et utilise JavaScript pour modifier dynamiquement le contenu.

  • SASS
  • JS
  • Custom properties
  • Utility classes

FrontendMentor Challenge

Traqueur d'IP

Un défi proposé par Frontend Mentor. Le défi consistait à créer une application de suivi des adresses IP et à faire en sorte qu'elle soit aussi proche que possible du design. L'application est construite avec react, utilise l'API de géolocalisation IP par IPify, et LeafletJS pour générer la carte. Les utilisateurs peuvent voir leur propre adresse IP au chargement initial de la page, et rechercher n'importe quelle adresse IP ou n'importe quel domaine et voir sa localisation et quelques informations supplémentaires.

  • SASS
  • JS
  • React
  • API
